# Inspire Me Now - Wellness & Mindfulness Platform

**Inspire Me Now** is a modern web-based wellness platform designed to support mental and emotional well-being. It offers users guided sessions, mindfulness exercises, and mood tracking tools, while providing administrators with a powerful dashboard for managing content and user activity.

---

## 🚀 Features

### 👤 User Management

* Admin can view and manage user details
* Session history, activity logs, and emotional data tracking

### 😊 Mood Tracking

* Users can log and monitor their emotional states
* Data visualizations to analyze emotional patterns over time

### 🎧 Guided Sessions

* Offers live and pre-recorded mindfulness and self-help sessions
* Includes affirmations, meditation, quick resets, and more

### 📱 Fully Responsive

* Works smoothly across desktop, tablet, and mobile devices

### 🧪 Demo Data Included

* Pre-filled sample data for quick demo and testing

---

## 📦 Installation

Follow the instructions below to set up and run the application locally.

### 🔑 Prerequisites

* **Node.js** v14.x or above
* **npm** v6.x or above **or** Yarn v1.x or above

### 📥 Clone the Repository

```bash
git clone https://github.com/azhar0i0/inspire-me-now.git
cd inspire-me-now
```

---

## 📚 Getting Started

### Install Dependencies

#### Using npm:

```bash
npm install
```

#### Using Yarn:

```bash
yarn install
```

### ▶ Run Development Server

#### Using npm:

```bash
npm start
```

#### Using Yarn:

```bash
yarn start
```

The application will now be available at:

```
http://localhost:3000
```

Open your browser to view the running app.

## 📁 Folder Structure

```
src/
├── assets/               # Images, logos, and other static assets
├── Components/           # Main components used across the platform
│   ├── Layout/           # Sidebar, Header, and layout elements
│   ├── Dashboard/        # Dashboard components and analytics
│   ├── UserManagement/   # User admin screens
│   ├── ContentManagement # Manage sessions, exercises, etc.
│   └── Parts/            # Reusable widgets (charts, cards, buttons, etc.)
├── Auth/                 # Authentication logic and context
├── App.js                # Root App component
├── index.js              # Application entry point
└── styles/               # Global stylesheets
```
